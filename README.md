# 🎮 Rock Paper Scissors Game (React)

## 📌 Project Overview

This is a simple and interactive **Rock-Paper-Scissors game** built using React. The user plays against the computer, which randomly selects its move. The app tracks scores, rounds, and displays the winner of each round.

---

## 🚀 Features

* 🧠 Play against computer (random moves)
* 📊 Score tracking (User vs Computer)
* 🔢 Round counter
* 🏆 Winner display after each round
* 📜 Move history (previous rounds)
* 🔥 Win streak tracker
* 🔄 Reset button to restart the game

---

## 🛠️ Tech Stack

* React (useState Hook)
* JavaScript (ES6)
* HTML & CSS

---



## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Navigate to the project folder:

```
cd rock-paper-scissors
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm start
```

---

## 🎯 How to Play

* Click on:

  * 🗿 Rock
  * 📄 Paper
  * ✂️ Scissors
* The computer will randomly choose its move.
* Scores and results update instantly.

---

## 🔄 Reset Game

Click the **Reset button (🔄)** to restart:

* Scores reset to 0
* Rounds reset
* History cleared
* Streak reset
